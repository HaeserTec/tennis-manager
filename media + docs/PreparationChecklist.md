# Preparation Checklist

## Overview
Use this checklist to ensure you're fully prepared for each session, reducing stress and improving efficiency.

## Pre-Session (Day Before or Morning Of)
- [ ] Review `WeeklySchedule.md` for the day's plan
- [ ] Check weather and court availability; have backup plan
- [ ] Prepare equipment: Balls (extra), cones, first aid kit
- [ ] Print/update player lists and progress notes
- [ ] Charge devices (phone for timer/video)
- [ ] Plan drills: Reference class README.md for ideas
- [ ] Send session reminders via `CommunicationTemplates.md`
- [ ] Personal prep: Hydrate, eat, rest

## During Session
- [ ] Arrive 30 minutes early to set up
- [ ] Warm-up players safely
- [ ] Monitor progress and take notes
- [ ] Adjust drills based on group dynamics
- [ ] Ensure safety: Watch for injuries, provide breaks
- [ ] Track attendance and payments

## Post-Session
- [ ] Update `PlayerProgress.md` with notes
- [ ] Clean up equipment and court
- [ ] Send follow-up communications (e.g., progress snippets)
- [ ] Reconcile finances in `Finances.md`
- [ ] Reflect: What worked? Improvements needed?
- [ ] Plan next session adjustments

## Weekly Review
- [ ] Update all files (Schedule, Progress, Finances)
- [ ] Order supplies if needed
- [ ] Review goals and adjust for next week
- [ ] Self-care: Schedule rest days

## Monthly Review
- [ ] Full reassessments and advancements
- [ ] Financial summary
- [ ] Parent feedback collection
- [ ] Long-term planning (e.g., season goals)

## Tips
- Check off items as completed.
- Add custom items for your setup.
- Use this to build habits and stay organized.
